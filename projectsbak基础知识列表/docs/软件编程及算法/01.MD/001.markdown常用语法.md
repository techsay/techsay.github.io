### markdown 插件安装：

支持流程图的插件：
npm install vuepress-plugin-mermaidjs@1.9.1 -D

```js
// .vuepress/config.js 配置
module.exports = {
    plugins: [
        'vuepress-plugin-mermaidjs'
    ]
}
```



Mermaid-组件示例：

``` mermaid
sequenceDiagram
Alice->John: Hello John, how are you?
loop every minute
    John-->Alice: Great!
end
```


使用一个主题和使用一个插件的方式几乎一致。
